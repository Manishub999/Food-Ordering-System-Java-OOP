# Food Ordering System in Java

## Overview
This is a console-based Food Ordering System developed using Java and Object-Oriented Programming (OOP) concepts. The project allows customers to browse food items, place orders, and calculate discounts for premium customers.

## Features
- Customer Management
- Premium Customer Support
- Food Item Management
- Menu Display
- Order Placement
- Discount Calculation
- Database Connectivity
- View Order Details

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- JDBC
- VS Code

## OOP Concepts Implemented
- Classes and Objects
- Inheritance
- Method Overriding
- Encapsulation
- Constructors
- Polymorphism

## Project Structure

- Customer.java
- PremiumCustomer.java
- FoodItem.java
- Menu.java
- Order.java
- DBConnection.java
- ViewOrders.java
- Main.java

## How to Run

1. Clone the repository
2. Open the project in VS Code
3. Compile all Java files
4. Run Main.java

Example:

```bash
javac *.java
java Main
```

## Sample Functionality

- Display food menu
- Select food items
- Place orders
- Calculate total bill
- Apply premium customer discount
- Store and retrieve order information

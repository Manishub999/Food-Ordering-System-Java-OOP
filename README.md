# Food Ordering System in Java

## Overview
Console-based Food Ordering System developed using Java, OOP, JDBC, and MySQL.

## Features
- Add and remove items from cart
- Budget-based suggestions
- Combo generation
- Premium customer discounts
- Admin login
- Order tracking
- Store and view orders using MySQL

## Technologies Used
- Java (JDK 17)
- OOP
- JDBC
- MySQL

## OOP Concepts
- Encapsulation
- Inheritance
- Polymorphism

## Project Structure
- FoodItem.java
- Menu.java
- Customer.java
- PremiumCustomer.java
- Order.java
- DBConnection.java
- ViewOrders.java
- Main.java
